PROFOUND GRID

A responsive grid system for fixed and fluid layouts.Built in SCSS,it gives you flexibility and full control

Responsive

make the grid fluid or fixed and add premidia layout as needed.Change the column and gutter widths and remove or add column as needed

Semantic

The grid keeps your markup sematic and works without any "grid_x" classes.unless you want to use them,in which case they are included.

Precise

Profound grid uses negative margins to calculate columns.Unlike other grid systems,fluid layout will look exactly the same in all browsers
